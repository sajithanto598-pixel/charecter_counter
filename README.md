# Character Counter

A simple and interactive Character Counter built using **HTML, CSS, and JavaScript**.  
This project allows users to type a message and track the number of characters used in real time.

The counter updates instantly as the user types and shows a warning when the maximum character limit is reached.

---

## 🚀 Features

- Real-time character counting
- Maximum character limit enforcement
- Dynamic UI updates using JavaScript
- Warning message when limit is reached
- Clean and simple user interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript 

---

## 📌 How It Works

1. The user types a message in the textarea.
2. JavaScript listens for the **input event**.
3. The script calculates the number of characters typed.
4. The counter updates automatically.
5. When the limit (200 characters) is reached, a warning message is displayed.
